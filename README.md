# GraphDefend

## Nosso projeto
Projeto acadêmico em desenvolvimento voltado ao estudo de teoria dos grafos. O GraphDefend analisa redes de computadores, identifica pontos críticos e utiliza o algoritmo para otimizar a conectividade com menor custo e maior segurança.
